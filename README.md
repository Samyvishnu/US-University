# US-University

Built a Data-Driven US University STEM Course Selection Assistant (307 Universities | 78 Disciplines)

Choosing the right undergraduate STEM program in the U.S. is overwhelming—especially for international students navigating unfamiliar systems, visa considerations, English requirements, and high financial stakes.

So I decided to solve it.

I built a comprehensive, interactive data product that simplifies UG STEM decision-making using real-world datasets and user-centric design.

🔍 What this project does:

Aggregates data across 307 U.S. universities and 78 STEM disciplines

Enables course-first discovery (e.g., Data Science, AI, Engineering, Biology)

Designed specifically to support international students with:

TOEFL/IELTS requirements

Conditional admission options

SAT/ACT waiver insights

Provides end-to-end filtering:

Tuition (COA), Net Cost, Scholarships

Acceptance Rates & SAT/ACT ranges

Graduation & retention outcomes

Includes a Score Matcher to categorize universities into Reach / Target / Safe

Offers side-by-side comparison and decision analytics

💡 Key Features I Designed:

📊 Advanced filtering + sorting across financial, academic, and admission metrics

🌍 Dedicated international student lens (language requirements, flexibility, accessibility)

🎯 Personalized SAT/ACT-based matching system

💰 Scholarship & affordability insights for better financial planning

🔬 STEM-first exploration (instead of generic university search)

⚖️ Multi-university comparison dashboard

🧠 Tech & Skills Applied:

Data modeling & transformation (multi-source datasets)

Frontend product design (interactive UI/UX)

Decision-support system thinking

Domain: Higher Ed, STEM pathways, admissions analytics

📈 Why this matters:
For international students, the U.S. application process can feel opaque and fragmented. This project turns scattered information into a single decision intelligence platform, helping students make confident, data-backed choices about their future.
